#### Constraints:  
+ `2 <=   coordinates.length <= 1000`  
+ `coordinates[i].length == 2`  
+ `-10^4 <=   coordinates[i][0],   coordinates[i][1] <= 10^4`  
+ `coordinates`   contains no duplicate point.
